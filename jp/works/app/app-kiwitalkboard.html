Kiwi Talkboard

App Development / PHP & MySQL

NZ移住者と現地を繋ぐ、安全でアクセシブルな掲示板アプリ

01. 作品概要

ニュージーランドでの生活や情報交換をスムーズにするための、階層型（スレッド式）掲示板アプリです。PHPとMySQLを用いたバックエンド処理と、モダンなUIを実現するTailwind CSSを組み合わせ、リアルタイムに近い感覚で投稿や返信ができる環境を構築しました。

02. 開発の背景と目的

2026年8月のニュージーランド移住計画を見据え、現地でのコミュニティ形成や情報収集に役立つツールを自作したいと考えたのがきっかけです。

03. リスク管理とセキュリティ

トレーサビリティのためのIPアドレス保存機能

目的： 匿名投稿による誹謗中傷やスパム投稿（荒らし行為）を抑制し、万が一のトラブル時に発信元を特定できる仕組みを構築しました。

実装内容： PHPの $_SERVER['REMOTE_ADDR'] を活用し、データベースに送信元IPを自動記録。バックエンド側で静かにセキュリティ性を高めています。

04. 使用技術（Tech Stack）

Frontend: HTML5, Tailwind CSS, JavaScript (Fetch API)
Backend: PHP 8.x
Database: MySQL (MariaDB)
Infrastructure: Lolipop Rental Server

05. こだわりのポイント

  • アクセシビリティ最優先の設計： 「誰一人取り残さない」という理念のもと、視覚的なコントラストの確保や、JavaScriptによるフィルタリング検索機能を実装しました。
  • 法的遵守への意識： 利用規約の同意確認やIP保存など、Webサービスを安全に運用するための法的・倫理的側面を考慮した設計を行いました。
